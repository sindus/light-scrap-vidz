Install

macOS & Linux — one command

Works on macOS (Apple Silicon) and Linux (Debian/Ubuntu/other).

curl -fsSL https://sindus.github.io/light-scrap-vidz/install.sh | bash
macOS: installs to /Applications/ from the official .dmg release.  ·  Linux (apt): installs the .deb via apt.  ·  Other Linux: installs an AppImage to ~/.local/bin/.

Uninstall

macOS
rm -rf /Applications/light-scrap-vidZ.app
Linux · Debian / Ubuntu
sudo apt remove light-scrap-vidz
Use sudo apt purge to also remove config files.
Linux · AppImage
rm ~/.local/bin/light-scrap-vidz

Requirements

  • macOS Apple Silicon (M1 or later)
  • Linux x86_64 — Debian 12+ / Ubuntu 22.04+ recommended; other distros via AppImage
  • No manual yt-dlp installation needed — it is bundled in the package
